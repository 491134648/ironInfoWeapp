# ironInfoWeapp
钢材信息小程序

# 技术选型
![](https://upload-images.jianshu.io/upload_images/2245742-aa8925cdb8e05969.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/579/format/webp)

# 使用的第三方库

 - ucharts
 - marked (md => html)
 - mpvue-wxparse (html => wxml)

# 目录介绍
```
├─common
├─api // 请求封装
├─components // 组件
│  ├─echarts
│  ├─marked
│  ├─mpvue-echarts
│  └─mpvue-wxparse
├─img // Readme用的图片文件夹
├─pages // 页面
│  ├─article // 文章页
│  ├─auth // 登录页
│  ├─details // 钢材详情页
│  ├─home // 首页
│  ├─index // 钢材搜索页
│  ├─iron // 钢材信息页
│  ├─search // 钢材搜索框
│  └─weather // 天气页
├─static // tabbar图片
└─store // vuex文件
```
